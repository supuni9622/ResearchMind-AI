# Local Development

> This guide explains how to set up a local ResearchMind development environment.

## Status

🚧 In Progress

This document will be updated throughout the project.

---

## Prerequisites

- Git
- Docker Desktop
- Python 3.12
- Node.js 22 LTS
- uv

---

## Development Environment

The local development environment consists of:

- FastAPI
- Next.js
- PostgreSQL
- Valkey
- Qdrant

Additional services will be added as the project evolves.

---

## Initial Setup

The current setup process is:

1. Clone the repository
2. Install Python using `uv`
3. Create the virtual environment
4. Install dependencies
5. Configure `.env`
6. Start infrastructure
7. Run the API

Detailed commands will be updated as each milestone is completed.
